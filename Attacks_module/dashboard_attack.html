Attacks User Interface

This Streamlit application provides an interactive user interface to simulate various cyberattacks against an IoT edge node running an anomaly detection system.

The supported attack vectors include: 1. Data Injection: Sending spoofed/impossible physical values. 2. Flooding (DoS): Overwhelming the broker with high-frequency messages. 3. Markov Process Tampering: Injecting logically impossible state transitions. 4. Replay Attacks: Re-transmitting old, captured telemetry sequences.
